\doxysubsection{Detailed Description}
\mbox{\hyperlink{classDiscoSquare}{Disco\+Square}} is a class that represents a square in the \mbox{\hyperlink{classDiscoPane}{Disco\+Pane}} grid. It extends the Java\+FX Rectangle class and implements the Runnable interface.
